# COVISART Advanced Technologies - Product Catalog

This is the official product catalog website for COVISART Advanced Technologies, showcasing the NGS-360-3 Next Generation Simulator.

## About

COVISART Advanced Technologies is a leading provider of next-generation simulation systems for defense and industrial applications. Our flagship product, the NGS-360-3, represents the pinnacle of simulation technology with complete 360-degree immersion.

## Technology Stack

- **Next.js 16** - React framework for production
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Google Fonts** - Exo 2 and Orbitron typography

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```
