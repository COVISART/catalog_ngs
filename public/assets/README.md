# Placeholder Images Guide

This directory contains placeholder images for the COVISART catalog website.
Replace these with actual product images from covisart.com.tr.

## Image Specifications

### Open Graph / Social Media Images
- **LinkedIn**: 1200×627 px (`/assets/og/linkedin-1200x627.jpg`)
- **Twitter**: 1200×600 px (`/assets/og/twitter-1200x600.jpg`)
- **Facebook/Meta**: 1200×630 px (`/assets/og/fb-1200x630.jpg`)
- **Instagram**: 1080×1080 px (`/assets/og/insta-1080x1080.jpg`)

### Hero Images
- **Main Hero**: 1920×1080 px (16:9 ratio) (`/assets/hero-placeholder-1920x1080.jpg`)

### Product Images
- **NGS-360-3**: 1920×1080 px (`/assets/ngs3603-blank-1920x1080.png`)
- **NGS-5000**: 1920×1080 px (`/assets/ngs5000-blank-1920x1080.png`)
- **NGS-Mini Trainer**: 1920×1080 px (`/assets/ngs-mini-blank-1920x1080.png`)

### Gallery Images
- **Gallery Items**: 1024×768 px (4:3 ratio) (`/assets/gallery/blank-1024x768.png`)

## How to Replace

1. Download actual images from covisart.com.tr
2. Resize to match the specifications above
3. Replace the placeholder files in the `/public/assets/` directory
4. Maintain the same filenames or update references in the code

## Notes

- All images should be optimized for web (compressed while maintaining quality)
- Use WebP format where possible for better performance
- Ensure images are representative of the actual products
- Update social media meta tags in `index.html` with correct image URLs
